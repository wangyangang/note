# css设置表格边框合并

```css
table{
            border:1px solid black;
            border-collapse: collapse;
            margin:0 auto;
            width:300px;

        }
        td{
            border:1px solid black;
            height:50px;
            width:100px;
            text-align: center;

        }
```
> table设置border-collapse为collapse即可。
